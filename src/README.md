# Лабораторная работа №5
## Использование циклов в Java

### Выполненные задания:
1. Вывод чисел от 1 до N (for)
2. Сумма чисел от 1 до N (while)
3. Факториал числа (for)
4. Чётные числа от 1 до 100 (while + continue)
5. Ввод чисел до 0 (do-while)
6. Таблица умножения (вложенные циклы)

## Запуск программы
Открыть файл Lab5.java и запустить метод main().
