Тема: Массивы в Java
Студент: Акрам Анваров

🎯 Цель работы

Изучить объявление, создание, инициализацию и обработку одномерных и двумерных массивов в языке Java.

⸻

🧩 Реализованные задания

Одномерные массивы
•	Вычисление суммы отрицательных элементов массива.
•	Вычисление суммы положительных элементов массива.
•	Нахождение произведения элементов по условию.
•	Вычисление среднего арифметического.
•	Обмен местами минимального и максимального элементов.
•	Сортировка массива по возрастанию.
•	Определение суммы чётных и нечётных элементов.

Двумерные массивы
•	Вычисление сумм элементов по строкам.
•	Вычисление произведений по столбцам.
•	Работа с главной диагональю матрицы.
•	Вывод таблицы случайных чисел.

Дополнительные задания
•	Реализация метода поиска максимального элемента.
•	Работа со строками и их объединением по условию.
