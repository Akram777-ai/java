# Лабораторная работа №4
## Работа с условными и циклическими операторами в языке Java

### Цель работы
Освоить принципы использования условных операторов и циклов в языке программирования Java.

### Задание
В рамках лабораторной работы реализованы программы для:
- анализа введённых пользователем данных;
- принятия решений с использованием условных операторов;
- выполнения различных вычислений и проверок.

### Реализованные примеры
1. Определение знака введённого числа
2. Определение оценки по введённому баллу (0–100)
3. Сравнение двух чисел
4. Определение дня недели по номеру
5. Проверка числа на чётность
6. Проверка доступа по возрасту
7. Определение количества дней в месяце
8. Анализ температуры
9. Проверка попадания числа в диапазон
10. Простой калькулятор

### Среда разработки
- Язык программирования: **Java**
- IDE: **IntelliJ IDEA**
- JDK: **17**
